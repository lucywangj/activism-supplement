Methodology


Data Sources

The data used in our analysis for all years proceeding 2015 was sourced directly from Mudd Library, which shared with the ‘Prince’ a database containing optical character recognition (OCR) data from the Larry Dupraz archive. We did not independently verify this database against the raw image data in the archive or Princeton University Library’s archive explorer. For years succeeding 2015, the ‘Prince’ used data from internal web archives.

Data Cleaning

We first removed all tokens of the form ’xx to prevent class year attributions from dominating frequency analysis. We removed all capitalization and punctuation in order to ensure that errors in OCR, which can frequently miscaptialize letters or insert unnecessary punctuation, would not prevent terms from being recognized. This means that all emblematic terms are not case sensitive, e.g. tokens like “princeton,” “Princeton,” “pRinceton” are all recognized as the same term.

Term Extraction

The ‘Prince’ used a whitespace tokenizer to count word frequencies for each word. Then, for each analyzed word, we compared the frequency of all terms in that word against a corpus containing data for that word and the preceding and following words. For the 2010s and 1880s, we used the single neighboring word. Like The Pudding, we only included words that were used at least once in 10,000 words in each word to ensure the relevancy of analyzed terms. Using each word’s list of disproportionately used words, we then searched the archives to find historically and culturally relevant uses of those words to share in the summaries.
